> ### How to build ```.env```
> 1. Go to .gitignore file
> 2. Add ```.env``` under ```# misc```
> 3. Create ```.env``` file in src folder
> 4. In ```.env``` file, type ```REACT_APP_<nameOfKeyInAllCaps>=<key>```
